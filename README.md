Helps everything (almost) of communications with Dataverse, or rather xRM as it was.

# What is `Rappen.AI.WinForm`?

This will help us all implement LLM AI features in the XrmToolBox or in any other application using `WinForm`.

Read details: https://jonasr.app/ai-code-helpers

# What is `Rappen.XRM.Tokens`?

This is a niche feature that dynamically populates data from Dataverse.

Read the doc: https://jonasr.app/xrm-tokens/

Read in wiki: https://github.com/rappen/Rappen.XTB.Helper/wiki/XRM-Tokens

# What is `Rappen.XTB.Helpers`?

All the help we need to simplify the development of tools in **XrmToolBox**, and a bunch of controls inherited from `WinForm` that can have Dataverse as a source (and target). Us it.

Read in wiki: https://github.com/rappen/Rappen.XTB.Helper/wiki/Controls
